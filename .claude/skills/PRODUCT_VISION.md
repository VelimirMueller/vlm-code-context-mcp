# Product Vision: vlm-code-context-mcp

## The Vision

An AI-powered virtual IT department that runs your entire development process through the Model Context Protocol. One npm package gives you: codebase intelligence, a 9-agent scrum team, full sprint lifecycle management, and an enterprise React dashboard — all orchestrated by Claude via 39 MCP tools.

## The Problem

AI agents waste context reading raw files. Development teams lack structured process. Sprint management lives in expensive SaaS tools disconnected from the codebase. There's no single system that gives AI agents both code understanding AND project management capability.

## The Solution

Pre-index your codebase into SQLite. Load a 9-role AI agent team. Run real sprints with planning, implementation, QA, and retrospectives — all through MCP. Every tool call is structured, every decision is tracked, every sprint produces measurable velocity.

## Who It's For

- Senior engineers managing AI-assisted development workflows
- Teams with 10K+ line codebases who want structured AI agent processes
- Organizations exploring AI-powered project management
- Solo developers who want enterprise-grade sprint discipline without the overhead

## Key Metrics

- 3x fewer context tokens per codebase query
- 8x less raw data transferred
- 19pt average sprint velocity across 12+ sprints
- 39 MCP tools covering code intelligence + scrum + persistence
- 9 AI agent roles with mood tracking and workload balancing

## Principles

1. **Zero-dependency intelligence** — metadata, not raw files
2. **Process as code** — sprint rules, retro findings, velocity trends — all in SQLite
3. **Agent autonomy with guardrails** — 8pt dev cap, mandatory retros, QA gates
4. **Everything persists** — dump/restore, descriptions survive re-indexing
5. **Ship every sprint** — 19pt target, real velocity tracking, no skipped retros
